Retro-synthesis in RetroBioCat

RetroBioCat features two complementary approaches to computer-aided synthesis planning:
Network explorer and Pathway explorer.

Network explorer offers ‘human-led’ computer aided retrosynthesis. Starting from the target molecule and working backwards, RetroBioCat can make suggestions for which reactions might be possible, allowing the user to direct the search for a synthesis route.

In contrast, Pathway explorer will automatically build a large number of potential pathways, and score these using a weighted score which can be set by the user. Pathways are then displayed in order of score, and can be navigated through one at a time.
